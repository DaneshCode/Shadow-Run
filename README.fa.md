# Shadow Run: کابوس بی‌پایان

Shadow Run یک بازی دوبعدی endless runner با فضای ترسناک است که با Python و Pygame ساخته شده. بازیکن به صورت خودکار به جلو می‌دود، بین زمین و سقف جاذبه را عوض می‌کند، به دشمن‌ها شلیک می‌کند، سکه جمع می‌کند و با قابلیت Ghost Mode برای مدت کوتاهی از خطر عبور می‌کند.

## ویژگی‌ها

- دنیای بی‌پایان با تولید مرحله به صورت رویه‌ای
- حرکت روی زمین و سقف با تغییر جاذبه
- سیستم تیراندازی با مهمات محدود و شارژ دوباره
- Ghost Mode شارژشونده برای نامرئی شدن موقت
- دشمن‌های زمینی، سقفی، پرنده، تعقیب‌کننده، تیرانداز، روح، تلپورتر، عنکبوتی، برزرکر و موجی
- امتیاز، سکه، سلامتی، پاورآپ و جدول امتیازهای ذخیره‌شونده
- سختی پویا بر اساس امتیاز
- پس‌زمینه پارالاکس، ذرات، لرزش دوربین و صدای محیطی واکنشی
- منوی اصلی، آموزش، توقف، تنظیمات، پایان بازی و لیدربورد

## کنترل‌ها

| عمل | کلیدها |
| --- | --- |
| پرش | `Space` / `W` / `فلش بالا` |
| تغییر جاذبه | `Shift` / `S` / `فلش پایین` / `F` |
| شلیک | `کلیک چپ` / `X` / `Z` / `Ctrl` |
| Ghost Mode | `کلیک راست` / `C` / `G` / دکمه HUD |
| توقف | `Esc` / `P` |

## نصب و اجرا

پیش‌نیازها:

- Python 3.8 یا جدیدتر
- Pygame

روی ویندوز:

```bash
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
python unigame.py
```

روی macOS یا Linux:

```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
python unigame.py
```

## ساختار پروژه

```text
Shadow-Run/
  unigame.py              نقطه شروع بازی
  requirements.txt        وابستگی‌های پایتون
  assets/                 صداها و تصویرها
  data/                   ذخیره لیدربورد
  docs/                   سند طراحی بازی
  game/                   کد اصلی بازی
```

## نکته‌های بازی

- Ghost Mode باید شارژ شود؛ وضعیت آن در دکمه دایره‌ای HUD دیده می‌شود.
- دشمن‌های تیرانداز هم روی زمین و هم روی سقف می‌توانند خطرناک باشند.
- موج دشمن‌ها در امتیازهای بالاتر فعال می‌شود.
- مهمات به آرامی برمی‌گردد، پس شلیک‌ها را برای مسیرهای خطرناک نگه دارید.

## بررسی سریع

```bash
python -m compileall unigame.py game
```
